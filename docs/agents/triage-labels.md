# Labels de triagem

| Papel canônico | Label no tracker | Significado |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Precisa de avaliação do mantenedor |
| `needs-info` | `needs-info` | Aguarda informações do autor |
| `ready-for-agent` | `ready-for-agent` | Está especificada e pronta para execução AFK |
| `ready-for-human` | `ready-for-human` | Exige implementação ou intervenção humana |
| `wontfix` | `wontfix` | Não será executada |

Quando uma skill mencionar um papel de triagem, use a label correspondente nesta tabela.
